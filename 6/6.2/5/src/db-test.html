Проверка подключения к БД

Примечание: На текущем этапе мы не можем выполнить реальную проверку подключения к базе данных из браузера напрямую из-за ограничений безопасности. В рабочем проекте такая проверка обычно выполняется через серверный код (PHP, Node.js, Python и т.д.).

Для проверки подключения рекомендуется использовать Adminer, доступный по адресу http://localhost:8080, с параметрами подключения, указанными на главной странице.
